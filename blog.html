JavaScript Concepts

Map is and iterative method and returns a new array, whilst forEach is just and iterative method but it returns undefined. Both of them does not mutate the original array. On the other hand, filter method filtrate elements of an array by given condition and returns a new array with the filtrate elements. Whereas, find methods returns the first matched item from the given array.

Local Storage stores data in the browser without any expiration as opposed to Session Storage which stores data in the browser for a session. The Data stored in Local Storage will never be deleted if it not being removed manually whereas Session Storage data deleted when Browser tab or browser is closed. Both of them store data in string format.

Conversely, Cookies data can be set with expire date and stored in both Browser and Server. It has only 4kb capacity.

Since, JavaScript is a single threated language, it can only execute one block of code at a time. Event loop is the mechanism of executing JavaScript code asynchronously. It has Call Stack which is a data structure used to keep track of function calls in a program. It also checks the Task Queue continuously for new tasks. Task Queue keeps track of functions that are going to be executed in the future. And Micro Task Queue which handle Promises. It keeps the main thread of execution and handle asynchronous tasks simultaneously.
